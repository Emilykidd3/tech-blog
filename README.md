# Tech Blog

## Purpose
A heroku deployed web tech blog that allows you to create a username and password to signin, post, edit your posts, and see other's posts. 

## Built With 
* HTML 
* CSS
* JavaScript
* Bcrypt
* Sequelize
* Express
* Mysql2
